# Dependencies

Uses the M2M_Logger library.

# Quectel modules

Quectel is a manufacturer of cellular modules.

This library supports the following modules.

 - [UG95](https://www.quectel.com/product/ug95.htm)
 - [M95](https://www.quectel.com/product/m95.htm)
